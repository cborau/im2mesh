get_mask_from_images¶

formatreader.get_mask_from_images(dirname, ext, z_size=30.0, n_interp=10, smooth_slices=True)¶

Retrieves the surface points of the 3D volume defined in multiple individual images. Images are automatically binarized via Otsu thresholding.

Parameters:

dirname –

The directory containing the images.

extstring

File extension (ej: .jpg, .png).

z_sizefloat

Vertical distance between slices.

n_interpint

Number of slices interpolated between every consecutive original images. Default = 10.

smooth_slicesbool

If True, original binarized images are smoothed via erosion/dilation operations. Default = True
